# 🐍 Py - Red Team Web Toolkit

Colección de herramientas de Red Team web escritas en Python para bug bounty y pentesting.

## Herramientas

| Herramienta | Descripción |
|-------------|-------------|
| `network_scanner.py` | Escáner de red local con detección de dispositivos, MAC, fabricante y puertos |
| `xss_scanner.py` | Scanner de XSS con 80+ payloads, soporte GET/POST/DOM |
| `directory_bruteforcer.py` | Fuerza bruta de directorios con 500+ rutas integradas y multihilo |
| `login_bruteforcer.py` | Fuerza bruta de logins con soporte multihilo y barra de progreso |
| `php_js_injector.py` | Detector + Explotador 2 en 1 con shell interactiva PHP/JS |
